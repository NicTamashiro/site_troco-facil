O Projeto

Uma Abordagem Sistemática para um Problema Clássico

O algoritmo recebe um valor monetário qualquer e devolve a menor lista de cédulas e moedas que compõem esse valor — tal como um caixa automático faria. O projeto integra validação de entrada robusta, aritmética inteira para evitar erros de ponto flutuante e saída legível para o usuário.

Desenvolvido na disciplina de Algoritmos da UNISANTOS, o trabalho explora fundamentos como divisão inteira, operador módulo e estruturas condicionais.

Por que aritmética inteira?

Trabalhar com centavos como inteiros evita os problemas de representação de ponto flutuante. Por exemplo, 0.1 + 0.2 ≠ 0.3 em muitas linguagens. Multiplicar por 100 e arredondar garante cálculo exato.

Validação de entrada Rejeita letras e valores ≤ 0
Algoritmo guloso Sempre usa a maior denominação possível primeiro
Sistema Real Brasileiro Cobre todas as 7 cédulas e 6 moedas vigentes
Passo a Passo

Como o algoritmo funciona

01
Entrada

Leitura e Validação

O programa solicita um valor ao usuário. Se a entrada contiver letras ou for ≤ 0, uma mensagem de erro é exibida e a execução é encerrada.

02
Conversão

Conversão para Centavos

O valor é multiplicado por 100 e arredondado para um inteiro (n). Isso elimina erros de ponto flutuante em todas as operações seguintes.

03
Cálculo

Divisão Gulosa

Para cada denominação (R$ 200 → R$ 0,01), calcula-se n ÷ denominação para obter a quantidade, e n mod denominação para o resto.

04
Notas

7 Cédulas Processadas

R$ 200, R$ 100, R$ 50, R$ 20, R$ 10, R$ 5 e R$ 2 são calculadas nessa ordem, sempre do maior para o menor valor.

05
Moedas

6 Moedas Processadas

R$ 1,00, R$ 0,50, R$ 0,25, R$ 0,10, R$ 0,05 e R$ 0,01 completam a decomposição do restante após as cédulas.

06
Saída

Exibição Seletiva

Apenas as denominações com quantidade > 0 são exibidas, produzindo uma saída limpa e objetiva, sem linhas desnecessárias.

Código Fonte

/* Decomposição Monetária — C */
#include <stdio.h>

int main() {
    float valor;
    printf("Digite o valor: ");
    if (scanf("%f", &valor) != 1) {
        printf("Erro: apenas números são permitidos.\n");
        return 1;
    }
    if (valor <= 0) {
        printf("Erro: valor deve ser maior que zero.\n");
        return 1;
    }

    /* Converte para centavos inteiros */
    int n = (int)(valor * 100 + 0.5);

    /* Notas */
    int notas200  = n / 20000; n %= 20000;
    int notas100  = n / 10000; n %= 10000;
    int notas50   = n / 5000;  n %= 5000;
    int notas20   = n / 2000;  n %= 2000;
    int notas10   = n / 1000;  n %= 1000;
    int notas5    = n / 500;   n %= 500;
    int notas2    = n / 200;   n %= 200;

    /* Moedas */
    int moedas1   = n / 100;   n %= 100;
    int moedas050 = n / 50;    n %= 50;
    int moedas025 = n / 25;    n %= 25;
    int moedas010 = n / 10;    n %= 10;
    int moedas005 = n / 5;     n %= 5;
    int moedas001 = n;

    /* Saída */
    if (notas200  > 0) printf("%d nota(s) de R$ 200,00\n", notas200);
    if (notas100  > 0) printf("%d nota(s) de R$ 100,00\n", notas100);
    if (notas50   > 0) printf("%d nota(s) de R$ 50,00\n",  notas50);
    if (notas20   > 0) printf("%d nota(s) de R$ 20,00\n",  notas20);
    if (notas10   > 0) printf("%d nota(s) de R$ 10,00\n",  notas10);
    if (notas5    > 0) printf("%d nota(s) de R$ 5,00\n",   notas5);
    if (notas2    > 0) printf("%d nota(s) de R$ 2,00\n",   notas2);
    if (moedas1   > 0) printf("%d moeda(s) de R$ 1,00\n",  moedas1);
    if (moedas050 > 0) printf("%d moeda(s) de R$ 0,50\n",  moedas050);
    if (moedas025 > 0) printf("%d moeda(s) de R$ 0,25\n",  moedas025);
    if (moedas010 > 0) printf("%d moeda(s) de R$ 0,10\n",  moedas010);
    if (moedas005 > 0) printf("%d moeda(s) de R$ 0,05\n",  moedas005);
    if (moedas001 > 0) printf("%d moeda(s) de R$ 0,01\n",  moedas001);

    return 0;
}
